Navigate a cyberpunk city's criminal underworld through four escalating missions вҖ”
crack passwords, decode cipher sequences, and navigate firewall mazes

## Features

- **Three distinct mini-games** вҖ” password cracker, sequence decoder (breach protocol), and a randomly-generated firewall maze
- **Trace meter** вҖ” every mistake inches you toward detection; hit 100% and the system locks you out
- **Branching narrative** вҖ” four story missions with unique flavour text, escalating difficulty, and a final boss with two separate endings
- **Tool system** вҖ” collect `port_scanner`, `vpn_shield`, and `root_kit` across missions; each carries forward and changes how future missions play out
- **Freelance contracts** вҖ” procedurally generated one-shot jobs with randomised clients, targets, and complications; three difficulty tiers
- **Local leaderboard** вҖ” top-10 high scores persisted to `data/leaderboard.json`; submit your alias after any run
- **Career tracker** вҖ” cumulative stats (story runs, freelance contracts, lifetime credits, endings) across all sessions
- **Save / load** вҖ” JSON-based game state persists your progress between sessions
- **Typewriter output** вҖ” all narrative text prints character-by-character; press Enter or Space to skip
- **Cyberpunk aesthetic** вҖ” green-on-black ANSI colour, Unicode box-drawing art, and per-system ASCII banners
- **Cross-platform** вҖ” pure stdlib; optional `colorama` for Windows legacy terminal support
- **Platform audio** вҖ” `winsound` beeps on Windows, `afplay` on macOS, silent on Linux

## Installation

```bash
# Clone the repository
git clone https://github.com/jamiemyers99/hacking-game
cd hacking-game

# No install required вҖ” pure Python 3 stdlib
python main.py

# Optional: install colorama for reliable colour on Windows legacy terminals
pip install colorama
python main.py
```

**Requirements:** Python 3.8 or later. No third-party packages required.

## Running Tests

The test suite uses Python's built-in `unittest` framework and requires no extra packages beyond `pytest` as a test runner.

```bash
cd hacking-game
python -m pytest tests/ -v
```

| Module tested           | What's covered                                                      |
|-------------------------|---------------------------------------------------------------------|
| `engine/player.py`      | Tool management, trace mechanics, level cap, serialisation round-trip |
| `engine/game_state.py`  | Save / load / delete with file isolation, corrupt JSON handling      |
| `engine/leaderboard.py` | Entry storage, sort order, MAX_ENTRIES cap, `is_high_score` logic   |
| `engine/career.py`      | All stat fields, ending tracking, back-fill of missing keys         |
| `minigames/`            | Grid dimensions, solvable-target constraint proof, maze BFS connectivity |
| `missions/freelance.py` | Briefing content/variation, system routing per difficulty           |

Tests run automatically on every push via GitHub Actions across Python 3.8, 3.10, and 3.12.

## How to Play

### Objective

You are a freelance hacker navigating a corrupt cyberpunk city.
Work through four missions вҖ” each connecting you to a progressively harder system.
Complete all four to expose the conspiracy at the heart of PROMETHEUS.

### The Trace Meter

```
  TRACE: [в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘] 40%
```

The trace bar fills every time you make a mistake. Reach **100%** and the target
system locks you out вҖ” mission failed. The bar resets after each successfully
completed mission, but **not** between the three stages of the final boss.

---

### Mini-Game 1 вҖ” Password Cracker

Guess a hidden password within a limited number of attempts.

| Difficulty | Word pool  | Attempts | Trace per wrong guess |
|------------|------------|----------|-----------------------|
| Easy       | 5-letter common words | 8 | +5  |
| Medium     | 6-letter technical words | 6 | +10 |
| Hard       | 8-letter hacker terms | 4 | +15 |

**Feedback:** After each wrong guess you receive Wordle-style feedback вҖ”
how many letters are in the correct position, and how many are in the word.

**Commands available at the guess prompt:**

| Input  | Effect                                                       |
|--------|--------------------------------------------------------------|
| `hint` | Reveals one random letter. Costs extra trace.               |
| `t`    | Opens the toolkit overlay (inventory). Does not use a guess. |

---

**Rules:**
- Even turns: pick any **column** from the highlighted row
- Odd turns: pick any **row** from the highlighted column
- The target sequence is always achievable вҖ” plan your path before committing
- Wrong picks cost trace but still advance the row/column state

**Commands available at the prompt:**

| Input | Effect                        |
|-------|-------------------------------|
| `t`   | Opens the toolkit overlay.    |

### Mini-Game 3 вҖ” Firewall Maze

Navigate from the top-left `()` to the bottom-right exit `><`
through a randomly generated maze. Trace increases every N steps.

```
  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ
  в–Ҳв–Ҳ()      в–Ҳв–Ҳ    в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ
  в–Ҳв–Ҳ  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ  в–Ҳв–Ҳ  в–Ҳв–Ҳ  в–Ҳв–Ҳ
  в–Ҳв–Ҳ      в–Ҳв–Ҳ      в–Ҳв–Ҳ  в–Ҳв–Ҳ
  в–Ҳв–Ҳв–Ҳв–Ҳ  в–Ҳв–Ҳв–Ҳв–Ҳ  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ  в–Ҳв–Ҳ
  в–Ҳв–Ҳ              ><  в–Ҳв–Ҳ
  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳ
```

| Key           | Action             |
|---------------|--------------------|
| `W` / `вҶ‘`     | Move up            |
| `S` / `вҶ“`     | Move down          |
| `A` / `вҶҗ`     | Move left          |
| `D` / `вҶ’`     | Move right         |
| `I`           | Open toolkit       |
| `Q`           | Abort connection   |

The maze is regenerated randomly every run вҖ” no two playthroughs are the same.

---

### Tools

Tools are unlocked by completing missions and carry forward for the rest of the game.

| Tool           | How acquired   | Effect                                                       |
|----------------|----------------|--------------------------------------------------------------|
| `port_scanner` | Mission 1 reward | Passive: grants a -15 trace buffer at the start of Mission 2 |
| `vpn_shield`   | Mission 2 reward | Single-use: halves all trace gain during Mission 3            |
| `root_kit`     | Mission 3 reward | Single-use: auto-solves one mini-game stage in Mission 4      |

Press `T` at any text input prompt during a mission to view your toolkit.

---

### Missions

| # | Title              | System                   | Mini-game          | Difficulty |
|---|--------------------|--------------------------|--------------------|------------|
| 1 | The Leak           | NexusCorp Internal       | Password Cracker   | вҳ…          |
| 2 | Follow the Money   | Iron Vault Financial     | Sequence Decoder   | вҳ…вҳ…         |
| 3 | Ghost Protocol     | Daedalus Defence Network | Firewall Maze      | вҳ…вҳ…вҳ…        |
| 4 | Zero Day *(boss)*  | PROMETHEUS AI Core       | All three in order | вҳ…вҳ…вҳ…вҳ…вҳ…      |

**Mission 4** runs all three mini-games back-to-back with no trace reset between
stages. On success, the player makes a final choice that determines one of two
distinct endings.
